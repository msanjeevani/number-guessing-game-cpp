# 🎯 Number Guessing Game in C++

A simple console-based **Number Guessing Game** developed in **C++**. The program generates a random number between **1 and 100**, and the player must guess the correct number with the help of hints.

## 📌 Overview

This project demonstrates the fundamentals of C++ programming, including random number generation, loops, conditional statements, user input handling, and problem-solving.

## ✨ Features

- 🎲 Generates a random number between **1 and 100**
- 💡 Provides hints:
  - **Too High! Guess Lower**
  - **Too Low! Guess Higher**
- 🔄 Allows unlimited attempts until the correct number is guessed
- 📊 Displays the total number of attempts
- 🖥️ Simple and interactive console interface

## 🛠️ Technologies Used

- **Language:** C++
- **Compiler:** GCC / G++
- **IDE:** Visual Studio Code

### Compile

```bash
g++ GuessNumber.cpp -o GuessNumber
```

### Run

**Windows**

```bash
GuessNumber.exe
```

**Linux/macOS**

```bash
./GuessNumber
```
